# London Tube Network Visualisation

A Python project developed as part of my Computing degree to visualise the London Underground network using graph theory and data visualisation techniques.

## Features

- Visual representation of London Underground stations
- Network connections between stations
- Graph-based analysis
- Data visualisation using Python libraries

## Technologies Used

- Python
- NetworkX
- Matplotlib

## Skills Demonstrated

- Graph theory
- Data visualisation
- Python programming
- Data structures and algorithms
- Problem solving

## Installation

Install the required libraries:

```bash
pip install networkx matplotlib
```

Run the program:

```bash
python tube_map.py
```
